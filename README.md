# Passgen -a simple password generator
シンプルなパスワード生成Chrome拡張。

## 使い方
桁数を選択後、英数字or英数字+記号ボタンを選択してパスワードを生成する。

## 技術スタック
- JavaScript
- React
- MUI
- Vite

## インストール方法
```sh
$ npm run build
```
上記コマンドを実行後、distフォルダーをChromeに読み込み.

## About Debug
```sh
$ npm run dev
```
